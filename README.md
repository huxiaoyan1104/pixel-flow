# Pixel Flow

> This repository contains public documentation. The extension source code is private.

Smart image workflow tool for Chrome. Inspect, organize, and archive web images with source context — all in the browser's side panel.

## What is Pixel Flow?

Pixel Flow is a Chrome extension that helps designers, marketers, researchers, ecommerce teams, and AI creators inspect images, organize assets, keep source records, and archive workflows — within the browser's side panel. It is built for repeatable visual research and asset archiving workflows, not only for one-time file saving.

## Core Features

- **Page image scanning** — Detect images on the current page and review them in the side panel
- **Smart filtering** — Filter by source, file name, format, size, aspect ratio, responsive image type, and tags
- **Library and tags** — Organize saved images with custom tags for projects, use cases, and review status
- **Metadata inspection** — Review dimensions, format, source context, EXIF data, and technical signals
- **AI fingerprint detection** — Parse AI-generation parameters: model, seed, sampler, CFG scale (99% confidence)
- **Controlled download and export** — Download images and export structured records with source URLs and context

## How is Pixel Flow different?

Most browser image tools are built for one-off saving. Pixel Flow is built for structured, repeatable workflows:

- Every image stays connected to its source URL and metadata
- Built-in tagging system for project organization
- Download history and export records for traceability
- AI fingerprint detection and AIGC parameter parsing
- User-controlled — no automatic scraping or background collection
