# Library and Export

The library stores images you actively save. Use tags to organize assets, multi-select for batch actions, and download history to review previous work.

## Tagging Ideas

- By project: `landing-page`, `spring-campaign`
- By use: `hero-image`, `product-detail`
- By status: `candidate`, `approved`, `needs-review`

## Before Exporting

Export records may include source URLs, tags, and context. Before sharing, check whether the file contains sensitive sources, client project names, or internal notes.
