# Feedback and Support

If you run into issues with scanning, downloading, sign-in, PRO access, or data management, use Contact Support in the settings page. Complete context helps support reproduce and resolve the issue faster.

## Helpful Information

1. Action path.
2. Affected page URL.
3. Expected result and actual result.
4. Whether the issue can be reproduced consistently.
5. Browser version, extension version, and system information.
6. For purchase or account issues, include the sign-in email or order reference.

Do not send passwords, verification codes, card numbers, or full payment card details.
