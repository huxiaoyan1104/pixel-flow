## v1 Feature Scope

Pixel Flow v1 focuses on web image scanning, filtering, preview, favorites, tags, metadata review, download history, and structured export. Format conversion is not part of the v1 feature set.

## Compliance Positioning

Pixel Flow is a user-controlled image productivity tool. It does not automatically scrape the open web or continuously collect website images in the background. Use it only where you are authorized to access, process, and reuse the relevant images.
