# AI Dataset Curation Best Practices

AI dataset work requires source awareness, permission checks, metadata review, and clear records. Pixel Flow helps you select candidate images and preserve structured context.

## Recommended Flow

1. Open only pages you are authorized to access and process.
2. Scan images on the current page.
3. Narrow candidates by dimensions, format, source, and tags.
4. Review metadata, AI fingerprint, and AIGC signals.
5. Tag purpose, source status, and review status.
6. Export records for later permission review, cleanup, and quality evaluation.

## Risk Reminder

Do not blindly collect web images. Dataset use, license scope, personal information, portrait rights, and website terms all require separate review.
