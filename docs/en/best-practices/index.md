# Best Practices

Best practices explain how different roles can use Pixel Flow more effectively. They focus on real tasks rather than button-by-button descriptions.

## Available Scenarios

- [Designers](./designers.md)
- [Ecommerce operators](./ecommerce.md)
- [Content teams](./content.md)
- [AI dataset curation](./ai-datasets.md)

## Recommended Template

1. Define the task.
2. Scan the current page.
3. Narrow results by format, dimensions, source, and tags.
4. Save candidate images.
5. Review details and metadata.
6. Batch download or export records.
7. Check authorization, privacy, and source context.
