# Designer Best Practices

Designers can use Pixel Flow as a visual reference workflow rather than a one-off image downloader.

## Recommended Flow

1. Open a reference site or competitor page.
2. Scan the current page images.
3. Filter out icons, avatars, and small thumbnails.
4. Preview candidate images and save useful references.
5. Add project tags such as `brand-refresh` or `hero-reference`.
6. Export records to preserve source and selection context.

## Filtering Tips

- Prioritize larger images.
- Use aspect ratio to find hero images, banners, or product visuals.
- Use tags to separate inspiration from assets that may be reused.

## Important Note

Reference does not mean permission. Before using images commercially, verify source, license scope, and client context.
