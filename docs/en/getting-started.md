# Install and Open

After installing the Chrome extension, open Pixel Flow from the extension action. The side panel is the main workspace for scanning, previewing, collecting, organizing, and downloading images.

## Recommended Flow

1. Install the extension and pin it to the browser toolbar if useful.
2. Open a page where you are authorized to access and process images.
3. Click the Pixel Flow extension action to open the side panel.
4. Start from the scan flow to detect images on the current page.

## Important Boundary

Pixel Flow actions are initiated by the user. Use scanning, saving, downloading, exporting, and reverse image search only where your use complies with site terms, copyright rules, and applicable law.
