# Scan and Filter

Open a page that contains images and use scan. Pixel Flow detects images on the current page and presents them in the side panel so you can review assets without repeatedly opening new tabs.

<figure>
  <img src="/images/docs/scan-page.webp" alt="Pixel Flow scan page in a browser window" />
  <figcaption>Scan results appear in the side panel with useful signals such as format, dimensions, and source context.</figcaption>
</figure>

## Filter Options

- Source and filename
- Image format
- Dimensions and size range
- Aspect ratio
- Responsive image behavior
- Custom tags

## Workflow Tip

Start with format and dimensions to remove obvious noise, then use source, tags, and preview to confirm the final candidate set.
