# Image Details and Metadata

Image details help you review dimensions, format, source context, available metadata, color, and technical signals. Some results are inferred from file content or metadata and should be treated as reference signals, not ownership or legal conclusions.

## Available Details

- **Dimensions and format** — Width, height, file type, and file size
- **Source context** — Original page URL, responsive image sources
- **Metadata** — EXIF data, camera parameters, creation timestamps
- **AI fingerprint** — Technical inference of AI generation likelihood
- **AIGC parameters** — Model, seed, sampler, CFG scale when available
- **Camera parameters** — EXIF, device, and shooting-related signals

## Usage Tip

Treat metadata as evidence to review, not a final answer. For licensing, copyright, portrait rights, or commercial reuse, verify the original source and authorization records.
