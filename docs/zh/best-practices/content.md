# 内容团队最佳实践

内容团队可以用 Pixel Flow 管理文章配图、活动素材、社媒视觉参考和下载记录。

## 推荐流程

1. 打开资料页、品牌页或素材页。
2. 扫描当前页面图片。
3. 过滤掉尺寸过小或来源不明的图片。
4. 收藏候选配图并按栏目、活动或发布日期打标签。
5. 导出图片记录，给编辑或设计复核。

## 标签建议

- `article-candidate`
- `campaign`
- `needs-license-check`
- `approved-source`

## 发布前检查

发布前确认图片授权、署名要求、肖像权和品牌使用限制。导出记录可作为内部复核清单，但不是授权文件。
