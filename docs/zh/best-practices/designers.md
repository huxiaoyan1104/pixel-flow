# 设计师最佳实践

设计师适合把 Pixel Flow 用作视觉参考整理工具，而不是一次性图片下载器。

## 推荐流程

1. 打开参考网站或竞品页面。
2. 扫描当前页面图片。
3. 过滤掉图标、头像和过小缩略图。
4. 预览候选图片，收藏真正有价值的视觉参考。
5. 使用项目标签整理，例如 `brand-refresh`、`hero-reference`。
6. 导出记录，保留来源和筛选上下文。

## 筛选建议

- 优先查看大尺寸图片。
- 用宽高比筛出 hero、banner 或商品图。
- 用标签区分“灵感参考”和“可复用素材”。

## 注意事项

参考不等于授权。用于商业项目前，需要确认素材来源、授权范围和客户使用场景。
