# 电商运营最佳实践

电商运营可以用 Pixel Flow 检查商品页图片、缩略图、响应式图片和下载记录，帮助整理商品素材和竞品页面观察结果。

## 推荐流程

1. 打开商品详情页。
2. 扫描页面图片。
3. 按尺寸、格式和来源筛选商品主图、详情图和缩略图。
4. 收藏候选素材，并添加商品或活动标签。
5. 查看图片详情，确认尺寸和可用元数据。
6. 导出记录给运营、设计或商品团队复核。

## 适合关注的信号

- 主图尺寸是否一致。
- 详情图是否存在多个版本。
- 缩略图和大图是否来自同一来源。
- 下载历史是否能追溯到具体页面。

## 注意事项

竞品图片通常不能直接复用。Pixel Flow 适合辅助检查和整理，不替代授权确认。
