# 素材库与导出

素材库保存你主动收藏的图片。你可以通过标签组织素材，用多选模式进行批量操作，并通过下载历史回看过往处理记录。

## 建议标签方式

- 按项目：`landing-page`、`spring-campaign`
- 按用途：`hero-image`、`product-detail`
- 按状态：`candidate`、`approved`、`needs-review`

## 多选模式

多选模式适合：
- 批量下载
- 批量导出
- 取消收藏
- 统一添加标签

## 导出前检查

导出记录可能包含图片来源、标签和其他上下文信息。分享前请检查是否包含敏感来源、客户项目名或不应公开的内部备注。
