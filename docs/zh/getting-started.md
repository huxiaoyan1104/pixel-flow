# 安装与打开

安装 Chrome 扩展后，可以从扩展入口打开 Pixel Flow 侧边栏。侧边栏是扫描、预览、收藏、管理和下载图片的主要工作区。

## 浏览器版本要求

Pixel Flow 基于 Chrome Extension Manifest V3 和 Chrome Side Panel API 构建。请使用 **Chrome 114 或更新版本**安装和运行；低于该版本可能无法安装扩展，或无法正常打开侧边栏。

## 推荐流程

1. 安装扩展并固定到浏览器工具栏。
2. 打开一个你有权访问和处理图片的网页。
3. 点击 Pixel Flow 扩展入口，打开侧边栏。
4. 从扫描流开始检测当前页面图片。

## 注意事项

Pixel Flow 的扫描、收藏、下载、导出和以图搜图都需要用户主动触发。请确保你的使用符合目标网站条款、版权规则和适用法律。
