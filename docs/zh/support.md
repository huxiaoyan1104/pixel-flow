# 反馈与支持

如果你遇到扫描、下载、登录、PRO 权限或数据管理问题，请通过设置页中的技术支持入口提交反馈。完整上下文能帮助更快复现和定位问题。

<figure>
  <img src="/images/docs/feedback.webp" alt="Pixel Flow 技术支持入口" />
  <figcaption>设置页中的技术支持入口会提示你如何提供问题上下文和联系邮箱。</figcaption>
</figure>

## 建议提供的信息

技术支持邮箱：[support@joinpixelflow.com](mailto:support@joinpixelflow.com)。

1. 操作路径。
2. 出现问题的网页地址。
3. 期望结果和实际结果。
4. 是否可以稳定复现。
5. 浏览器版本、扩展版本和系统信息。
6. 与购买或账号有关的问题，请提供登录邮箱或订单信息。

请不要发送密码、验证码、银行卡号或完整支付卡信息。
