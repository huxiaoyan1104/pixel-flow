## v1 功能范围

Pixel Flow v1 聚焦网页图片扫描、筛选、预览、收藏、标签、元数据查看、下载历史和结构化导出。格式转换不在 v1 功能范围内。

## 审核与合规定位

Pixel Flow 是用户主动控制的图片生产力工具。它不会自动抓取全网内容，也不会在后台替用户持续采集网站图片。请只在你有权访问、处理和复用相关图片的合法授权范围内使用。
