# 扫描与筛选

打开包含图片的页面后，使用扫描功能。Pixel Flow 会检测当前页面图片，并在侧边栏中集中展示，减少反复打开新标签页的操作。

## 可以筛选什么

- 来源和文件名
- 图片格式
- 尺寸和规格
- 宽高比
- 响应式图片类型
- 自定义标签

## 工作建议

先用格式和尺寸排除明显无关图片，再用来源、标签和预览确认最终素材。对于电商、设计参考和内容配图工作，建议每次只保留与当前任务相关的一组图片。
